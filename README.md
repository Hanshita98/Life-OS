# LifeOS – Personal Discipline System

A Flutter mobile app for building discipline across studies, health, spirituality, habits, and productivity.

## Features

- Daily routine tracker with completion percentage.
- Habit tracker with streak and consistency.
- Study tracker with daily/weekly stats.
- Spiritual routine tracker (prayer, meditation, gratitude).
- Health tracker (steps, water, exercise).
- Goals section (long-term, monthly, daily).
- Distraction control with warning indicators.
- Dashboard with daily score out of 100 and motivation quotes.
- Morning checklist, night reflection, and weekly review prompts.
- Life area score cards (Health, Study, Spirituality, Money).

## Tech Stack

- Flutter (cross-platform)
- SQLite via `sqflite`

## Run

```bash
flutter pub get
flutter run
```
